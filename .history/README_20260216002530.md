# 宮廟小編神器 🏯

協助宮廟人員快速生成單頁官網與社群宣傳圖的免費線上工具

## 🌟 功能特色

- **📱 單頁官網生成** - 快速建立宮廟簡易官網，包含地址、電話、Google Map 連結
- **🕯️ 點燈宣傳圖** - 製作光明燈、太歲燈等點燈服務宣傳圖
- **🎉 神明誕辰海報** - 設計神明聖誕祝壽活動海報
- **📢 公告圖片** - 產生春節時間、活動通知等公告圖

## 🚀 線上使用

直接開啟網頁即可使用，無需安裝任何軟體：
[https://你的GitHub帳號.github.io/templetry/](https://你的GitHub帳號.github.io/templetry/)

## 💡 使用方式

1. 選擇功能模式（官網/點燈/誕辰/公告）
2. 填寫宮廟資訊
3. 上傳圖片（可選）
4. 調整圖片位置與大小
5. 點擊下載按鈕

## 📦 技術架構

- **前端框架**: Alpine.js
- **樣式**: Tailwind CSS
- **圖片處理**: html2canvas
- **部署**: GitHub Pages

## 🔧 本地開發

```bash
# 直接開啟 index.html 即可
# 或使用簡易伺服器
python -m http.server 8000
```
